# Data Science Assignment
This project is completed as part of the ExcelR Data Science training program. The objective of this assignment is to apply fundamental data science concepts on a real-world dataset to extract meaningful insights and support data-driven decision making.

The project involves multiple stages of the data science workflow, including data collection, data cleaning, exploratory data analysis (EDA), and data visualization. During data preprocessing, missing values were handled, duplicate records were treated, and data types were corrected to ensure data quality.

In the exploratory data analysis phase, statistical summaries and visual patterns were analyzed to understand relationships between variables. Various Python libraries such as Pandas and NumPy were used for data manipulation, while Matplotlib and Seaborn were used for visualization to identify trends, distributions, and correlations.

Key insights were derived from the dataset to understand patterns and behavior within the data, which can further help in business decision-making. This assignment demonstrates practical understanding of data handling, analysis techniques, and visualization skills using Python.
